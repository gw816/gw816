这个最好是有硬件的支持，提供HDMI的输出，如果要实现像91或者豌豆夹的效果的话，其实很简单，就是读取framebuff的内容，不断的通过adb传输到pc上，简单的传输可以通过adb push 到PC本地，然后解析为bitmap，效率高的可以使用adb +socket 。 还可以在pc屏幕上将鼠标坐标转换为触屏坐标，通过adb模拟按键发送到手机端，实现PC对手机的控制，例如点击（）：
如在屏幕的x坐标为40，y坐标为210的点上touch一下，命令如下
adb shell sendevent /dev/input/event0 3 0 40
adb shell sendevent /dev/input/event0 3 1 210

adb shell sendevent /dev/input/event0 1 330 1 //touch
adb shell sendevent /dev/input/event0 0 0 0 //it must have

adb shell sendevent /dev/input/event0 1 330 0 //untouch
adb shell sendevent /dev/input/event0 0 0 0 //it must have

以上可能会与不同手机设置有点出入，event0需测试得出

对于逢培训、会议、宣讲会必带笔记本做大屏投影的职场通病，想必很多像我一样的职场人，一边做着笔记本搬运工的苦力一边还要做培训。曾经我就想过，关于手机这种方便携带的移动设备，能不能像笔记本一样进行PPT投影的问题。以下内容就是本人探索使用手机进行PPT大屏投影的测试分享，废话比较多，请各位看官多加担待。


首先，手机能不能进行PPT大屏投影？

通常手机在微信上直接打开PPT，几乎是一页长文到底，只能下拉，画面有种紧凑感，如果在投影仪上看几乎是不能实现电脑版的PPT效果。


本人用的是vivo手机，但是因为工作文件比较多，所以我的手机上是安装了WPS办公软件，每次在微信接收到PPT文件，直接打开是翻不了页的而只能下拉观看，但是如果通过WPS软件，其实是可以实现自动播放PPT的，如下图：


既然手机能实现大屏投影，那么手机能不能进行PPT大屏投影？经过本人测试其实是可以的，而且还能让手机PPT在投影仪的展示就跟笔记本投影的PPT模式一样。而这个实现过程就需要在手机上安装WPS（主要是确保手机上打开的PPT效果是横屏分页的），外加投影需要的手机HDMI接口转换器和HDMI接口数据线。


手机PPT大屏投影需要哪些配件？

手机实现PPT大屏投影，其需要手机充电器（充电头和数据线）、HDMI接口的多功能转换器和HDMI接口的数据线即可，由于公司配置了HDMI接口的数据线，所以本人就只准备了绿联的一款多功能转换器和转换器配置的USB数据线，先来张高逼格的产品图压压惊，中间那个像MP3的货就是用到的转换器。


手机和转换器如何完成PPT大屏投影？

1、首先在手机上下载一个名叫家庭多媒体分享软件的“EZCast”，手机应用市场直接搜索即可安装；

2、由于这款转换器需要通电才能使用，所以在通电的时候，我直接使用vivo手机的充电器给其通电，当转换器通上电的时候，会有明显的指示灯亮起，而这也说明转换器已经成功充上电了；

3、进行接口连接，主要连接是HDMI接口和投影仪连接、手机跟转换器的连接（安卓手机数据线即可），再跟投影仪连接起来，最后还要用原装数据线连接手机完成整个接口连接。


4、一切准备工作做好后，无需账号手机直接登录EZCast，在EZCast页面中（如下图中的步骤1、2、3），先点击右上角的数据线连接，然后安卓手机点击“Connect”—选择“USB共享网络”；

5、完成USB共享网络后，就可以打开镜像了，由于镜像支持安卓5.0系统，刚好本人的手机就是5.0系统的，所以投影仪就可以读取手机里的视频、文件等，打开镜像步骤如下：

6、通常本人会给自己的微信号发培训要用的PPT文件，所以在微信里用WPS软件打开PPT之后，投影仪就会同步手机PPT培训内容。

备注：在整个使用的过程中，连接转换器的手机始终处在通电的过程中，这就能保证在PPT演示过程中不会因为手机没电而导致培训中断。另外在整个过程中，手机上会保持亮屏确保投影仪不会造成黑屏，所以这就能很好的保证整个会议或者培训的PPT的顺利完成。


个人经常用手机做大屏的PPT投影，感觉比较很快速，一旦第一次安装了EZCast，以后直接登录就可以操作了，比电脑大屏投影方便了很多。其实除了能给PPT大屏投影，其实还可以大屏播放视频和手机照片等功能；除了手机大屏投影，其实绿联这款多功能转换器也支持平板和笔记本电脑的大屏投影。
