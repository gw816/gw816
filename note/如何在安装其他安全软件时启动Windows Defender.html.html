Windows Defender 是一款内置于Windows Vista及以上版本的安全防护软件，但是在Windows 8/8.1/10里才是真正的安全防护软件，在

Windows Vista/7里面只是一款间谍防护软件。其实，只要是Windows 7/Vista用户，如果在安装了微软的MSE以后，然后再看看Windows 

8/8.1/10的Windows Defender，就不难发现，MSE的界面和Windows 8/8.1/10的Windows Defender几乎一模一样。

如果是Windows 8/8.1/10的用户，我们不难发现在安装其他安全软件以后，Windows Defender 就会自动关闭，以减少内存占用，但是我不建

议关闭Windows Defender，而且从某种角度说，Windows Defender要比其他的国产软件好得多，因为其实国内的安全软件的使用效果都差不

多，但是我们用户用360、腾讯电脑管家、金山和百度杀毒+卫士等等的人群也是比较多的，那么如何在不卸载这些安全软件的情况下，启动

Windows Defender呢？

方法/步骤

  1. 1

    退出您安装的安全防护软件

    安全软件启动的时候通常都会在任务栏的通知区域显示图标，请右击这些安全软件的图标，然后点击诸如“退出”的选项。并在退出的提示

    框里点击“确定”。

    如果系统的操作中心没有提示有关病毒防护和间谍防护软件的提示，请重新打开安全软件，设置不开机启动、不启动服务、关闭主动防御服

    务等等项目，然后重新启动计算机。

    *必须退出所有的安全软件后才能继续下一步操作。

  2. 2

    在操作中心打开Windows Defender

    重新启动计算机以后，如果弹出用户账户控制说某个安全软件要更改计算机，请单击取消。

    通常如果操作中心检测到所有的安全软件关闭，会在通知区域警告用户，让用户选择安全软件启动，这时请选择Windows Defender，然后单

    击“启用”。如果没有提示所有安全软件被关闭，请重复步骤一。

  3. 3

    设置组策略

    前两部算是基本完成了启动Windows Defender，但是我们也希望其他安全软件启动怎么办？注意，这个时候如果启动其他安全软件，就会前

    功尽弃——你会发现Windows Defender再次被关闭。

    所以，请打开组策略，方法：按下Win+R（Windows 8/8.1/10）或者在开始菜单的徽标右键（Windows 10里有开始菜单，操作和Windows 7 类

    似），选择运行（R），然后在打开（O）的输入框输入gpedit.msc。

    *一定要加上.msc，因为gpedit.msc是微软的通用管理文档文件，不是EXE，如果省略Windows 会提示无法找到文件。

    找到“关闭Windows Defender”配置项，然后在对话框中点击“已禁用”，点击“确定”。

    打开命令提示符（管理员）

    方法：在开始菜单的徽标右键，选择命令提示符(管理员)（A）。如果弹出用户账户控制，请输入管理员密码或者许可程序运行。
