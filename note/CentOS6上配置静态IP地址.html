CentOS6上配置静态IP地址

时间:2014-10-01 22:36来源:csdn 作者:Think-througher 举报 点击:3602次

问题

默认情况下，CentOS 6安装没有网络接入相关配置，需要你对网络接口进行配置。

解决方案

查看缺省接口配置

接口相关配置保存在/etc/sysconfig/network-scripts目录下，下面显示的是缺省的接口配置。
[plain] view plaincopy
 
  1. [root@CentOS65 network-scripts]# more /etc/sysconfig/network-scripts/ifcfg-eth0   
  2. DEVICE=eth0  
  3. HWADDR=00:0C:29:C1:48:03  
  4. TYPE=Ethernet  
  5. UUID=b2bbb718-caa6-46c3-ae12-587cbedfd5fa  
  6. ONBOOT=no  
  7. NM_CONTROLLED=yes  
  8. BOOTPROTO=dhcp  

配置接口静态地址

[plain] view plaincopy
 
  1. # vi /etc/sysconfig/network-scripts/ifcfg-eth0  
  2.    
  3. DEVICE="eth0"  
  4. NM_CONTROLLED="yes"  
  5. ONBOOT=yes  
  6. HWADDR=A4:BA:DB:37:F1:04  
  7. TYPE=Ethernet  
  8. BOOTPROTO=static  
  9. UUID=5fb06bd0-0bb0-7ffb-45f1-d6edd65f3e03  
  10. IPADDR=10.220.33.108  
  11. NETMASK=255.255.255.0  

配置网关

[plain] view plaincopy
 
  1. # vi /etc/sysconfig/network  
  2.    
  3. NETWORKING=yes  
  4. HOSTNAME=centos6  
  5. GATEWAY=10.220.33.1  

 重启网络接口

[plain] view plaincopy
 
  1. [root@CentOS65 network-scripts]# /etc/rc.d/init.d/network restart  
  2. Shutting down interface eth0:  [  OK  ]  
  3. Shutting down loopback interface:  [  OK  ]  
  4. Bringing up loopback interface:  [  OK  ]  
  5. Bringing up interface eth0:  Determining if ip address 10.220.33.108 is already in use for device eth0...  
  6.   
  7. [  OK  ]  

配置DNS服务器

[plain] view plaincopy
 
  1. # vi /etc/resolv.conf  
  2.    
  3. nameserver x.x.x.x # Replace with your nameserver ip  
  4. nameserver y.y.y.y # Replace with your nameserver ip  

查看接口配置

使用"ip a"或ifconfig命令可以查看当前接口配置信息
