-3 打开控制台(cmd)，运行下好的补丁IdeaCrackv2.1.jar或者IdeaPatch.jar，即输入java -jar IdeaCrackv2.1.jar（如果是IdeaPatch.jar则为 java -jar IdeaPatch.jar）命令，这时候，idea的控制台会让你输入一个路径。

-4 由于已经将idea.jar放入d盘下，所以直接复制D:\idea.jar这个路径即可，略等片刻，
