企业如何无人值守快速安装win 8
企业计算机管理员在给企业的很多电脑装系统时，如何能同时给多台电脑同时装呢，本文就介绍其中的一个比较简单、方便又快速的安装方法。
本方法所需工具为：
1、Windows评估和部署工具包 (Windows ADK) For Windows 8 下载地址为：http://www.microsoft.com/en-us/download/details.aspx?id=30652
2、UltraISO 下载地址大家自己百度哦
3、win 7 时代使用的iso镜像直接制作成u盘安装版的工具 Windows 7 USB DVD Tool

1、安装下载好的Windows ADK，程序首先会进行解压缩，得到三个文件，双击运行ADKinstall.exe，进行部署工具的安装。
