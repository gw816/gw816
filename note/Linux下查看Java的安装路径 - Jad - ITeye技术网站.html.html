Linux下查看Java的安装路径

  • 博客分类：
  • Linux
  • Ubuntu
 

 

Linux下查看Java的安装路径用命令 whereis java就可以看到了

eg:

username@ubuntu:~$ whereis java

java: /usr/bin/java /etc/java /usr/lib/java /usr/share/java /opt/jdk1.6.0_01/bin/java

 

如果仅仅想知道java执行路径在那里的话 用which java就可以了

eg:

username@ubuntu:~$ which java

/usr/bin/java
